# 🛒 Amazon Clone (HTML, CSS & JavaScript)

A fully responsive front-end clone of the Amazon e-commerce website built using **HTML, CSS, and JavaScript**.  
This project replicates the core UI/UX design and essential interactive features of the original Amazon platform for learning and portfolio purposes.

---

## 📌 Project Overview

This project focuses on:

- Structuring complex layouts using HTML  
- Styling with modern CSS (Flexbox, Grid, Responsive Design)  
- Implementing dynamic functionality using JavaScript  
- Improving DOM manipulation and event handling skills  
- Understanding e-commerce website structure  

---

## 🚀 Features

- 🏠 Homepage with product categories  
- 🔍 Search bar UI  
- 🛍️ Product listing section  
- 🛒 Add to Cart functionality (Frontend-based)  
- 🧾 Cart page with dynamic item updates  
- 📱 Fully Responsive Design  
- 🎨 Amazon-like UI Styling  
- 🔄 Interactive elements using JavaScript  

---

## 🛠️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling & Layout  
- **JavaScript (Vanilla JS)** – Interactivity & DOM Manipulation
